1.Install XAMPP.

2.Create DB.
	Database name : userdetails

3.Create Table.
	Table name : userdata

4.Create Fields.
	Fields name : userId(auto increment, primary key)
		      userName(varchar 255)
		      userPhoneNumber(varchar 255)
		      userEmailId(varchar 255)
		      userAddress(varchar 255)

5.Open project with your editor.

6.Run command inside project folder.
			php -S localhost:8000
